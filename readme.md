## Практическое занятие 2

Семантическая HTML-вёрстка сайта по макету Avion.

**Технологии:** HTML5, CSS, Flexbox(Header, NewsLetter), Grid(Benefits, Products, Popular, Footer)

В проекте используется методология **БЭМ (Блок–Элемент–Модификатор)**, которая обеспечивает логичную структуру CSS-классов, улучшает читаемость кода и упрощает его поддержку и масштабирование. Использование БЭМ позволяет легко масштабировать проект и вносить изменения без риска повлиять на другие элементы интерфейса.
